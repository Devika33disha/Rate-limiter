<h1>Rate Limiter System</h1>

<p>A simple Python OOP project that limits how many requests a user can send in a fixed time window.</p>

<h2>Features</h2>
<ul>
  <li>Takes user name as input first</li>
  <li>Lets the user send requests</li>
  <li>Blocks requests after the limit is reached</li>
  <li>Shows request count in the current time window</li>
  <li>Built using only Python and OOP concepts</li>
</ul>

<h2>Classes Used</h2>
<ul>
  <li><b>User</b> - stores user name and request history</li>
  <li><b>RateLimiter</b> - checks whether a request is allowed or blocked</li>
  <li><b>RateLimiterSystem</b> - handles the full menu-driven program</li>
</ul>

<h2>How It Works</h2>
<p>The program asks for the user name first. After that, the user can send requests through a simple menu. If the number of requests crosses the allowed limit within the given time, the system blocks further requests.</p>

<h2>Tech Used</h2>
<ul>
  <li>Python</li>
  <li>Object-Oriented Programming (OOP)</li>
</ul>

<h2>Purpose</h2>
<p>This project was made to practice OOP in Python by building a simple real-world system.</p>
